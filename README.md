🛒 E-commerce Customer Behavior Analysis Dashboard
📊 Project Overview

This project analyzes customer behavior on an e-commerce platform to understand how users interact with the website, what factors influence purchases, and which channels drive the most revenue.

The analysis combines Python (Pandas) for data cleaning and validation and Power BI for interactive dashboard visualization. The final dashboard highlights key performance metrics, customer journey patterns, and marketing effectiveness to support data-driven business decisions.

🎯 Objective

The goal of this project is to:

Analyze customer interaction patterns

Identify top performing marketing channels

Understand device usage trends

Analyze product category revenue contribution

Examine conversion funnel performance

Study the relationship between session duration and purchase behavior

📂 Dataset

The dataset represents user sessions on an e-commerce website and contains information about:

Device type used by the customer

Marketing channel that brought the visitor

Product category viewed

Session duration

Cart activity (added to cart / abandoned)

Purchase status

Revenue generated

The dataset was cleaned and validated using Python and Pandas before being used in Power BI.

📈 Key Metrics (KPIs)

The dashboard tracks the overall performance of the platform using the following KPIs:

Total Revenue: $3,450,780

Total Sessions: 982,145

Purchase Rate: 3.85%

Average Order Value: $91.20

These metrics provide a quick overview of business performance and customer purchasing behavior.

📊 Dashboard Insights
Revenue Trend

Monthly revenue visualization helps identify sales patterns and seasonality throughout the year.

Revenue by Product Category

Electronics generates the highest revenue, followed by Clothing and Home & Kitchen, indicating strong demand in these categories.

Marketing Channel Performance

Conversion analysis across channels shows that:

Social Media has the highest purchase rate

Email marketing performs strongly

Referral traffic has the lowest conversion rate

This insight helps businesses optimize marketing budgets.

Device Usage Distribution

The analysis shows a significant share of traffic coming from mobile devices, highlighting the importance of mobile-optimized user experience.

Customer Conversion Funnel

The funnel visualizes the customer journey:

Visited → Added to Cart → Purchased

A noticeable drop occurs between Add to Cart and Purchase, indicating potential cart abandonment issues.

Purchase Rate vs Session Duration

The analysis shows how time spent on the website affects purchase probability, helping businesses understand user engagement behavior.
